## Описание проекта

Продвинутая ML-система для прогнозирования итоговой стоимости страхового возмещения (**UltimateIncurredClaimCost**) на основе датасета **Workers Compensation** (OpenML ID: 42876).

Датасет содержит 100 000 страховых дел с информацией о возрасте, зарплате, занятости, описании травмы и других характеристиках застрахованных.

### Задача

**Регрессия:** предсказать итоговую стоимость страховой выплаты по страховому случаю.

## Реализованные задачи

### Задача 0 — Сегментация данных
- Автоматическая кластеризация через **K-Means** (2–6 сегментов)
- Статистический анализ по сегментам
- Сравнение: общая модель vs специализированные модели для каждого сегмента

### Задача 1 — Детальный анализ данных
- Описательная статистика (100 000 строк, 14 признаков)
- Распределение целевой переменной (оригинал + log-преобразование)
- Корреляционный анализ
- Анализ категориальных признаков (boxplot по группам)
- Scatter: начальная оценка vs итоговая стоимость

### Задача 2 — Feature Engineering
Создано **8+ новых признаков**:
| Признак | Описание |
|---------|----------|
| `Age_x_WeeklyPay` | Взаимодействие возраста и зарплаты |
| `Estimate_per_Pay` | Отношение начальной оценки к зарплате |
| `HasDependents` | Бинарный: есть иждивенцы |
| `IsFullTime` | Бинарный: полная занятость (≥35 ч/нед) |
| `Log_InitialEstimate` | Лог-преобразование начальной оценки |
| `Accident_Year/Month/DayOfWeek` | Временны́е признаки из даты |
| `Reported_Year/Month/DayOfWeek` | Временны́е признаки из даты подачи |
| `ReportDelay_Days` | Задержка подачи заявки (дней) |

### Задача 3 — Обучение мощных моделей
Сравнение 4 моделей на log-преобразованной целевой переменной:
- **Random Forest** (100 деревьев)
- **Gradient Boosting**
- **XGBoost**
- **LightGBM**

Метрики: R², RMSE, MAE с интерактивным bar-chart сравнением.

### Задача 4 — Оптимизация гиперпараметров (Optuna)
- Оптимизация **XGBoost** с помощью Optuna (10–100 итераций)
- Пространство поиска: n_estimators, max_depth, learning_rate, subsample, colsample_bytree
- График истории оптимизации
- Сравнение базовой vs оптимизированной модели

### Задача 5 — ClearML интеграция
- Создание версионированного датасета
- Эксперимент 1: Linear Regression baseline
- Эксперимент 2: XGBoost + Optuna
- Логирование метрик, графиков, гиперпараметров

### Задача 6 — Интерпретируемость (SHAP)
- **Summary Plot (bar):** топ-признаки по влиянию на предсказания
- **Beeswarm Plot:** детальное распределение SHAP-значений
- **Waterfall Plot:** объяснение отдельного предсказания

### Задача 7 — Анализ ошибок
- Топ-20 случаев с наибольшей ошибкой
- Распределение абсолютных ошибок
- Scatter: факт vs прогноз
